Everyday Chef API Documentation

This is the backend API for the Everyday Chef app.

Important details:

  • Uses Edamam API for recipe data.

Search

Returns a list of recipes for a specific query

Path: https://everydaychef-api.herokuapp.com/recipes

Parameters

Parameter Required Description
q Yes Query text. For example q=tacos

Example URL: https://everydaychef-api.herokuapp.com/recipes?q=tacos

Post

Create a new recipe to add to the database.

Path: https://everydaychef-api.herokuapp.com/recipes

Parameters

Parameter Required Description
label Yes Title of recipe. For example label=Fish Tacos
image Yes Image URL for displaying on client. For example image=https://www.edamam.com/web-img/b3a/b3ac43e915f8a866a939e809f551eb3c.jpg
url No URL for original recipe source. For example url=http://www.lottieanddoof.com/2009/07/picadillo/
yield No The number of servings. For example yield=3.5
totalTime No The amount of time to cook recipe in hours. For example totalTime=1.5
ingredientLines Yes An array of required ingredients.

Example Post Body:
{ "label": "Fish Tacos",
"image": "https://www.edamam.com/web-img/b3a/b3ac43e915f8a866a939e809f551eb3c.jpg",
"url": "http://www.lottieanddoof.com/2009/07/picadillo/",
"yield": 3.0,
"totalTime": 1.5,
"ingredientLines": [
"Fresh Tilapia",
"Flour Tortillas",
"Salsa"
] }
